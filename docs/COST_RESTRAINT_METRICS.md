# Cost Restraint Metrics

EMEX must not reward faster over-testing.

## Initial Synthetic Signals

- Cost/restraint cautions parsed from OE output.
- Duplicate or low-value testing cautions preserved in structured suggestions.
- Broad workup language flagged for clinician review.

## Later Real-Data Metrics

- Orders per patient.
- Labs per patient.
- Imaging per patient.
- CT rate.
- Duplicate tests avoided or identified.
- Low-yield tests not recommended.
- Patient cost exposure where available and approved.
