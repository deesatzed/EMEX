# Discordance Review Plan

Discordance review is a later after-the-fact analysis layer.

## Buckets

- EMEX/OE suggested; provider ordered immediately.
- EMEX/OE suggested; provider ordered later.
- EMEX/OE suggested; provider never ordered.
- Provider ordered; EMEX/OE did not suggest.
- Provider had data unavailable at T0.
- Triage data incomplete.
- Guideline or local workflow ambiguous.

Observed outcomes judge safety. Knowledge-base review judges appropriateness only.
